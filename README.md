## Description
```bash
An Nest API and React App Auth web app
```
Feature list handles
- **API - NestJS with TypeScrip, TypeORM and MySQL database**
  - User
    - Login
    - Sign-up
    - JWT
    - Refresh token

- **FE - React JS, TypeScript, Material UI, QueryClient**
  - User
    - Login
    - Sign-up
    - Sign-out
